# Flutter Self-Study Roadmap

This repository documents my Dart and Flutter self-study journey based on a 6-week, 30-session roadmap.

## Goals

- Learn Dart fundamentals before going deeper into Flutter.
- Build small, focused practice outputs in each session.
- Keep notes, screenshots, and code to show steady progress.
- Finish with a small Flutter capstone project that is demo-ready.

## Study Flow

For each session, I try to keep the same loop:

1. Read only the required docs for that session.
2. Rebuild the example by typing the code myself.
3. Adjust the example a little to confirm I understand it.
4. Write a short learning note.
5. Commit code or screenshots as proof of progress.
