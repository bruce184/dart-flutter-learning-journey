# Phase 1: Beginner

This phase builds the practical foundation for the rest of the journey: environment setup, core Dart, Flutter basics, layout, forms, navigation, networking, simple state, and a first capstone.

## Modules

- `01_setup_dart_basics/`: environment setup and hand-written Dart practice
- `02_flutter_fundamentals/`: first Flutter app, widget tree basics, and app structure awareness
- `03_layout_forms_navigation/`: UI composition, forms, validation, and routing fundamentals
- `04_networking_state_basics/`: API calls, JSON parsing, UI states, and beginner-friendly state flow
- `05_beginner_capstone/`: small capstone app that combines the first phase skills

## Current Focus

The existing work in this repo is currently concentrated in:

- `01_setup_dart_basics/exercises/`
- `02_flutter_fundamentals/first_app/`

The remaining modules are scaffolded and ready to fill as the beginner roadmap continues.
